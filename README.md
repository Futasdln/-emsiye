# KAFM Origin: DRYSTATION ☔

An interactive, high-performance 3D product showcase for the **KAFM Origin DryStation** - a conceptual next-generation umbrella sterilization and drying system. Built with React Three Fiber, emphasizing corporate ROI, fluid physics, and Apple-tier manipulative marketing UI.

![Preview](https://img.shields.io/badge/Status-Production_Ready-success) ![Tech](https://img.shields.io/badge/Tech-React_Three_Fiber-000000?logo=react) ![Build](https://img.shields.io/badge/Build-Vite-646CFF?logo=vite)

## ✨ Özellikler (Features)

*   **Fluid Fluid Dynamics:** Gerçekçi ışık kırılması (refraction 1.33 IOR), su yüzeyi çalkalanma fiziği (sloshing) ve dinamik oksijen kabarcıkları (micro-bubbles).
*   **Volumetric Lighting:** Şemsiyeyi yıkayan mor UV-C lazer hüzmesi (Additive Blending) ve hareketli iç mekanik fan aydınlatmaları.
*   **Corporate ROI Dashboard:** AVM ve kurumlar için satın alma algısına oynayan animasyonlu CSS / HTML yatırım-geri-dönüş matrisleri (Sıfır kaza riski, Maksimum temizlik tasarrufu).
*   **Optimized WebGL Pipeline:** Cihazları çökertmeyen `renderOrder` kilitleri ve %100 "Siyah Ekran" koruması.

## 🚀 Kurulum & Çalıştırma (Quick Start)

Bu proje **Vite** ve **React** tabanlıdır. Kodu VS Code'a veya herhangi bir editöre indiren herkes saniyeler içinde çalıştırabilir.

1. **Bağımlılıkları Yükleyin:**
   Projeyi VS Code ile açtığınızda terminali (`Ctrl` + `~`) başlatın ve tüm gerekli paketleri kurun:
   ```bash
   npm install
   ```

2. **Geliştirme Sunucusunu Başlatın:**
   Hemen ardından projeyi tarayıcınızda canlı görmek için aşağıdaki komutu girin:
   ```bash
   npm run dev
   ```

3. Gelen adrese tıklayın (örn: `http://localhost:5173`) ve 3D deneyimin tadını çıkarın!

## 📦 Üretime Hazırlık (Build for Production)

Projeyi gerçek bir sunucuda (Vercel, Netlify, cPanel vb.) yayınlamak istiyorsanız optimize edilmiş statik dosyaları çıktısını almanız gerekir:
```bash
npm run build
```
Bu işlem `dist` adında sıkıştırılmış ve yayına hazır bir klasör oluşturur.
